Basics of Linux

What is Linux?

Linux is a powerful, secure, and open-source operating system created by Linus Torvalds. It has many distributions that are specifically designed for different purposes such as development, servers, security, and daily desktop use.

Features of Linux:

  • Linux is a powerful operating system.
  • It is open source and can be installed on any computer.
  • It is very secure and stable.
  • We can create a bootable USB drive to run Linux.
  • Linux is widely used in the IT industry.

Linux Distributions

Ubuntu

Ubuntu is a user-friendly, secure, and open-source operating system. It is widely used in development, servers, and cloud computing.

Kali Linux

Kali Linux is designed for cybersecurity and penetration testing. It comes with pre-installed hacking and forensic tools.

Zorin OS

Zorin OS is based on Ubuntu and provides a Windows or macOS-like interface, making it suitable for beginners.

Debian

Debian is one of the oldest and most stable Linux distributions. It is widely used on servers and is the base for many other distributions like Ubuntu.

Arch Linux

Arch Linux is a lightweight and flexible rolling-release distribution for advanced users who prefer full system control.

Fedora

Fedora is a cutting-edge Linux distribution sponsored by Red Hat. It is commonly used for development and testing new Linux technologies.

Linux Mint

Linux Mint is based on Ubuntu and Debian. It is popular for desktop users because of its simplicity, speed, and multimedia support.

Red Hat Enterprise Linux (RHEL)

RHEL is a commercial Linux distribution used in enterprise environments, servers, and cloud platforms with professional support.

Manjaro Linux

Manjaro is based on Arch Linux but is more user-friendly. It offers Arch features with easier installation and maintenance.

Elementary OS

Elementary OS is known for its clean and beautiful design inspired by macOS. It is ideal for everyday desktop use.

Installation of Linux

  1. Download the ISO file of the Linux distribution you want to install.
  2. Install the Rufus software. It is used to create a bootable USB drive.
  3. Open the Rufus software.
  4. Insert your USB pendrive into the computer.
  5. Select the USB device (it is usually selected automatically).
  6. Select the downloaded ISO file.
  7. Choose persistence if you want to save changes while using live Linux.
  8. Click the Start button. This process will format your USB drive and may take a few minutes.
  9. Restart your computer and press the boot menu key (usually F11 or F12).
  10. Select the USB device from the boot menu.
Linux will start in live mode, and your changes will be saved if persistence is enabled.
