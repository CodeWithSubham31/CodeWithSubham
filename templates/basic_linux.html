Basic of Linux

What is Linux?

Linux is powerful, secure, open source operating system. Linux made by Linus Torvalds. Linux have many distribution specificly devide for your tasks

Features of Linux :

  • Linux is powerful OS
  • It's Open source. You can install it any Computer
  • It's very secure OS
  • We can make a bootable pendrive of Linux
  • Linux used in IT field

Linux Distributions

Ubuntu

Ubuntu is a user-friendly, secure and open-source OS. It is widely used in development, servers, and cloud computing.

Kali Linux

Kali Linux is designed for cybersecurity and penetration testing. It comes with pre-installed hacking and forensic tools.

Zorin OS

Zorin OS is based on Ubuntu with a Windows/Mac-like GUI, suitable for beginners shifting from other OS platforms.

Debian

Debian is one of the oldest and most stable Linux distributions. It is used in servers and forms the base for many other distros like Ubuntu.

Arch Linux

Arch Linux is a lightweight, flexible, rolling-release OS for advanced users who prefer to build their system from scratch.

Fedora

Fedora is cutting-edge, secure, and sponsored by Red Hat. It is mainly used for development and testing the latest Linux features.

Linux Mint

Linux Mint is based on Ubuntu and Debian. It is popular for desktop users because of its simplicity, speed, and multimedia support.

Red Hat Enterprise Linux (RHEL)

RHEL is a commercial Linux distribution used in enterprise environments, servers, and cloud platforms with professional support.

Manjaro Linux

Manjaro is based on Arch Linux but more user-friendly. It is great for desktop users who want Arch features with easier setup.

Elementary OS

Elementary OS is known for its beautiful and minimalistic design, inspired by macOS. It is focused on everyday desktop use.

Installation of Linux

  1. Install the ISO file of the linux distribution that you want
  2. Install rufus software. This software make a USB pendrive to bootable.
  3. Run the rufus software.
  4. Insert your pendrive.
  5. Select the pendrive name. It will usually selected.
  6. Select the ISO file of the linux distro.
  7. Chose the persistancy. This persistancy needed to store any changes that you done in the bootable pendrive linux.
  8. Click on start button. It will take time. The data you have stored in the pendrive it will formated.
  9. After installing the ISO in pendrive restart your computer and click the bootmenu key. It usually F11 and F12.
  10. Select the pendrive name.
The Linux will start in live pendrive and every changes will stored
