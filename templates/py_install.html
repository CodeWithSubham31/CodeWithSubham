Installation & Getting Started

Steps to Install Python:

  1. Visit the official python website
  2. Download the executable file based on your OS
  3. Run the installer and complete installation
Version:

After installation, check the version by typing:

python --version

Starting Python:

Open Python IDE or any text editor. Type the following:

print("Hello World !!!")

Now save the file with a .py extension and run it. You will get the following output:

Hello World !!!

Installing Packages:

To install packages in Python we use the pip command. e.g.

pip install "Package Name"

Example (pandas):

pip install pandas
