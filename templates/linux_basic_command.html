Basic linux commands

whoami

This command returns the username that you set while you booting the os.

Example : 
whoami
uname

This command returns the name of your os.

Example : 
uname
passwd

This command used to change the password of your linux distro.

Example : 
passwd
sudo su

This command used to activate the super user mode.

Example : 
sudo su
date

This command return time and date.

Example : 
date
uptime

This command return time.

Example : 
uptime
history

This command return all the command that you used previously

Example : 
history
pwd

Print working directory

Example : 
pwd
clear

This command clear the terminal screen

Example : 
clear
apt-get update

This command updates your system

Example : 
sudo apt-get update
apt-get upgrade

This command upgrade your system

Example : 
sudo apt-get upgrade
reboot

This command shutdown your linu distro and get you into the main os.

Example : 
reboot
man

This command gives information about any command

Example : 
man cd
sleep

This command shut down your terminal for some seconds.

Example : 
sleep 5

Now your terminal will shutdown for 5 seconds.
