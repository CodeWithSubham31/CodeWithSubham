Basic Linux Commands

whoami

This command displays the currently logged-in username.

Example:
whoami
uname

This command displays information about your operating system.

Example:
uname
passwd

This command is used to change your Linux user password.

Example:
passwd
sudo su

This command switches to superuser (root) mode.

Example:
sudo su
date

This command displays the current date and time.

Example:
date
uptime

This command shows how long the system has been running.

Example:
uptime
history

This command displays the list of previously executed commands.

Example:
history
pwd

This command prints the current working directory path.

Example:
pwd
clear

This command clears the terminal screen.

Example:
clear
apt-get update

This command updates the package list from repositories.

Example:
sudo apt-get update
apt-get upgrade

This command upgrades installed packages to the latest version.

Example:
sudo apt-get upgrade
reboot

This command restarts the system.

Example:
reboot
man

This command displays the manual (help) page of any command.

Example:
man cd
sleep

This command pauses execution for a specified number of seconds.

Example:
sleep 5

The terminal will wait for 5 seconds before continuing.
