In python we use try...except block for error handeling. The code in the try block runs when there is no error. If the try block catches an error, then the except block is executed.

Example : 
try:
    num = int(input("Enter an integer: "))
except ValueError:
    print("Number entered is not an integer.")
Output1 : 
Enter an integer: 6.022
Number entered is not an integer.
Output2: 
Enter an integer: -8.5
Number entered is not an integer.
Output3: 
Enter an integer: -36

In addition to try and except blocks, we also have else and finally blocks. The code in the else block is executed when there is no error in the try block.

Example : 
try:
    num = int(input("Enter an integer: "))
except ValueError:
    print("Number entered is not an integer.")
else:
    print("Integer Accepted.")
Output : 
Enter an integer: 69
Integer Accepted.

The finally block is executed whatever the outcome of try... except... else blocks.

Example : 
try:
    num = int(input("Enter an integer: "))
except ValueError:
    print("Number entered is not an integer.")
else:
    print("Integer Accepted.")
finally:
    print("This block is always executed.")
Output : 
Enter an integer: 19
Integer Accepted.
This block is always executed.
