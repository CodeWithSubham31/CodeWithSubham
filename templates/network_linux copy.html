Network management commands

ifconfig

This command returns the ip address of your command.

Example : 
ifconfig
ip a

This command is same as ifconfig but it highlight the main information.

Example : 
ip a
netstart

This command used to check the network connection.

Example : 
netstart
iwconfig

This command used to config that wireless network interfaces.

Example : 
iwconfig
nslookup

This command used to get the ip address of any target

Example : 
nslookup google.com
arp

This command used to check the ip, mac address of your device.

Example : 
arp -a
route

This command used to check open port of your device.

Example : 
route
ping

This command used to check any website is active or deactive.

Example : 
ping youtube.com
tracepath

This command used to get the ip address of a website

Example : 
tracepath -4 google
