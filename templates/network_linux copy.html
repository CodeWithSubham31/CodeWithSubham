Network Management Commands

ifconfig

This command displays network interface information, including IP address, MAC address, and status.

Example:
ifconfig
ip a

This command shows detailed and modern network interface information. It is the replacement for ifconfig.

Example:
ip a
netstat

This command displays active network connections, listening ports, and routing information.

Example:
netstat -tulnp
iwconfig

This command is used to configure and display wireless network interface information.

Example:
iwconfig
nslookup

This command is used to find the IP address of a domain name using DNS.

Example:
nslookup google.com
arp

This command displays the ARP table, showing IP and MAC address mappings of connected devices.

Example:
arp -a
route

This command displays the routing table used to determine how network packets are forwarded.

Example:
route -n
ping

This command checks whether a host or website is reachable and measures network latency.

Example:
ping youtube.com
tracepath

This command shows the network path (hops) taken by packets to reach a destination.

Example:
tracepath google.com
