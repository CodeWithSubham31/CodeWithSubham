Python Operators

Python has different types of operators for different operations. They are as follows:

Arithmetic Operators

Arithmetic operators are used to perform arithmetic/mathematical operations.

Operator Name Operator Sign Example
Addition+5 + 3 = 8
Subtraction-9 - 4 = 5
Multiplication*6 * 2 = 12
Division/10 / 2 = 5
Modulus%10 % 3 = 1
Exponentiation**2 ** 3 = 8
Floor Division//10 // 3 = 3

Comparison Operators

These operators are used to compare values.

Operator Name Operator Sign Example
Equal to==5 == 5 → True
Not Equal to!=5 != 3 → True
Greater than>7 > 3 → True
Less than<2 < 5 → True
Greater than or Equal to>=6 >= 6 → True
Less than or Equal to<=4 <= 8 → True

Logical Operators

These operators are used to deal with logical operations.

Operator Name Operator Sign Example
Logical ANDand(5 > 2) and (4 > 1) → True
Logical ORor(5 < 2) or (4 > 1) → True
Logical NOTnotnot(5 > 2) → False

Assignment Operators

These operators are used to assign values to variables.

Operator Name Operator Sign Example
Assign=x = 5
Add and Assign+=x += 3
Subtract and Assign-=x -= 2
Multiply and Assign*=x *= 4
Divide and Assign/=x /= 2
Modulus and Assign%=x %= 3
Exponent and Assign**=x **= 2
Floor Div and Assign//=x //= 2

Bitwise Operators

Bitwise operators are used to deal with binary operations.

Operator Name Operator Sign Example
Bitwise AND&5 & 3 = 1
Bitwise OR|5 | 3 = 7
Bitwise XOR^5 ^ 3 = 6
Bitwise NOT~~5 = -6
Left Shift<<5 << 1 = 10
Right Shift>>5 >> 1 = 2
