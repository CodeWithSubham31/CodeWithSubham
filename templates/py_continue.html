Continue Keyword

What is Continue keyword?

The continue keyword in Python is used inside loops (for or while) to skip the remaining code in the current iteration and immediately move to the next iteration of the loop.

  • The loop starts running
  • When continue is reached → Python ignores the remaining code in that loop cycle.
  • Then the loop goes back to the beginning and checks the condition for the next cycle.
Example:
for i in range(5):
    if i == 2:
        continue  # Skip when i = 2
    print(i)
Output : 
0
1
3
4
