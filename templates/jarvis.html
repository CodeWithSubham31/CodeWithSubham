In Iron man movie we all saw the ai named Jarvis. Jarvis was most intelligent ai assistant made by Tony Stark. In this project we will make a A.I. assistant like Jarvis.

Features of this A.I.

  1. Voice Feature
  2. Speech Recognition Feature
  3. Chating Feature
  4. Time showing Feature
  5. Open any webaite Feature
  6. Open any system application Feature
  7. Youtube and google searching Feature
  8. Getting any images Feature
  9. Getting any topic information from wikipedia Feature
  10. Weather forcasting Feature
  11. Send messages Feature
  12. Joke Feature
  13. Text translating Feature

How we can make it?

  • We can use input() function for getting input from user.
  • We can use Infinity loop for getting inout every time.
  • We can use pyttsx3 module for AI voice feature
  • We can use pyaudio and speech-recognition module for speech recognition feature.
  • We can use time or datetime module for getting the current time.
  • We can use if-elif-else block for for user knowing what user want.
  • We can use webbrowser module for opening website, youtube and google search.
  • We can use os module for open system application.
  • We can use wikipedia module for fetch data from wikipedia.
  • We can use pywhatkit module for send whatsapp messages.
  • We can use deep_translator module for text translating.
