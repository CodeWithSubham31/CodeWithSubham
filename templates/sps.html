Stone paper secior game

Many of them we play the game named Stone Paper Secior. So yours project will be the Stone paper secior game.

Rules of this game :

  1. If 1st player chose stone and 2nd player chose paper, so the 2nd player will win the game.
  2. If 1st player chose stone and 2nd player chose secior, so the 1st player will win the game.
  3. If 1st player chose paper and 2nd player chose stone, so the 1st player will win the game.
  4. If 1st player chose paper and 2nd player chose secior, so the 2nd player will win the game.
  5. If both players chose same the match will be draw.

How we can make?

  • Use random for chosing random guess.
  • Use input() function for getting the user guess.
  • Use loops for getting the input from user multi times.
  • Use if-elif-else block for calculate you win or loss or draw.
