Python Input

The input() function in Python is used to take input from the user. It always returns the input as a string.

Syntax : 
input(prompt)
Simple Example of input()

In this code input()get your name as input and return hello with your name. 

name = input("What is your name : ")
print("hello ",name)
Output : 
What is your name : subham 
hello subham
Basic calculator using input()

We can use int() for convert string number to integer

num1 = int(input("Enter 1st number : "))
num2 = int(input("Enter 2nd number : "))
print(num1 + num2)
Output : 
Enter 1st number : 10
Enter 2nd number : 20
30
