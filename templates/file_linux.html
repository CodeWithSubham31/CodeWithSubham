File Handling Commands

ls

This command shows all files and folders in the current directory.

Example:
ls
ls -a

This command shows all files including hidden files and folders.

Example:
ls -a
cd

This command is used to change the directory.

In the terminal, folders are called directories.

Example:
cd Desktop
cd ..

This command takes you to the previous directory.

Example:
cd ..
touch

This command creates a new file in the current directory.

Example:
touch file.txt
echo

This command prints the text written after the echo command.

Example:
echo hello world

This command can also be used to create a file.

Example:
echo hello world > file2.txt

This will create a file named file2.txt containing "hello world".

cat

This command is used to read the contents of a file.

Example:
cat file2.txt
cp

This command is used to copy a file to another directory.

Example:
cp file.txt Downloads
mv

This command is used to rename or move a file.

Example:
mv file.txt file3.txt
rm

This command is used to delete a file.

Example:
rm file.txt
mkdir

This command creates a new folder in the current directory.

Example:
mkdir folder1
rmdir

This command deletes an empty folder.

Example:
rmdir folder1
rm -rf

This command deletes a folder and all its contents permanently.

Example:
rm -rf folder1
tree

This command displays files and folders in a tree structure.

Example:
tree
free

This command shows information about system memory usage.

Example:
free
ps

This command displays the list of currently running processes with their IDs.

Example:
ps
kill

This command is used to stop a running process using its process ID.

Example:
kill 1234
nano

Nano is a built-in text editor in Linux used to create and edit files.

Example:
nano file.txt
more

This command is used to read large files page by page.

Example:
more file.txt
less

This command is similar to more but faster and more flexible.

Example:
less file.txt
head

This command shows the first few lines of a file.

Example:
head file.txt
tail

This command shows the last few lines of a file.

Example:
tail file.txt
sort

This command sorts the contents of a file alphabetically.

Example:
sort file.txt
