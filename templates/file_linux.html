File handeling commands

ls

This command show all the files and folders that stord in a folder.

Example : 
ls
ls -a

This command show all the hidden files and folders

Example : 
ls -a
cd

Change Directory.

In Terminal we called folder as directory

Example : 
cd Desktop
cd ..

This command return you to the previous directory

Example : 
cd ..
touch

This command create a file in your working directory.

Example : 
touch file.txt
echo

This command print any text that you write after echo command.

Example : 
echo hello world

This command also used to create files.

Example : 
echo hello world > file2.txt

Now this command will create a file in your current working directory that have hello world.

cat

This command used to read a file

Example : 
cat file2.txt
cp

This command used to cpy a file to another directory.

Example : 
cp file.txt Downloads
mv

This command used to rename a file.

Example : 
mv file.txt file3.txt
rm

This command used to remove a file.

Example : 
rm file.txt
mkdir

This command create a folder in your working directory.

Example : 
mkdir folder1
rmdir

This command remove a empty folder.

Example : 
rmdir myself
rm -rf

This command used to remove a folder are not empty

Example : 
rm -rf myself
tree

This command return a tree shaped structurs of your file anf folder

Example : 
tree
mmv

This command used to move a file to another folder.

Example : 
mmv file.txt Downloads
free

This command returns all the information about your computer memory.

Example : 
free
ps

This command returns all the active process list with the id

Example : 
free
kill

This command used to kill any process that are running

Example : 
kill process id
nano

nano is a prebuilt text editor for linux. You can create files using this software

Example : 
nano file.txt
more

This command used to read a big files

Example : 
more file.txt
less

This command also used to read a big files. But this command is faster than more.

Example : 
less file.txt
head

This command show first lines of a file

Example : 
head file.txt
tail

This command show that last lines of a file.

Example : 
tail file.txt
sort

This command used to sort a file

Example : 
sort file.txt
