What is Datatype?

Data type specifies the type of value a variable requires to do various operations without causing an error. By default, Python provides the following built-in data types:

Numeric Data : int, float

  • int : 10,20,11,2
  • float : 10.0,20.0,11.0,2.0
Example : 
num1 = 10#type int
num2 = 20.0#type int

print(num1)#output : 10
print(num2)#output : 20.0

Text data : str

  • str : "hello world","python is best"
Example : 
name = "subham"
planet = "earth"

print(name)#output : subham
print(planet)#output : earth

Boolean data : True, False

  • True : In programming True means yes.
  • False : In programming False means no.
Example
a = True
b = False

print(a)#output : True
print(b)#output : False

Structured Data : list, tuple, set

  • list : ["subham","shayan","biltu","rocky","ronni"]
  • tuple : ("subham","shayan","biltu","rocky","ronni")
  • set : {"subham","shayan","biltu","rocky","ronni"}

    • Lists Data

    A list is an ordered collection of data with elements separated by a comma and enclosed within square brackets. Lists are mutable and can be modified after creation.

    Example : 
    names=  ["subham","shayan","biltu","rocky","ronni"]#creating list
print(names)
    Output : 
    ["subham","shayan","biltu","rocky","ronni"]
    Tuple Data

    A tuple is an ordered collection of data with elements separated by a comma and enclosed within parentheses. Tuples are immutable and cannot be modified after creation.

    Example : 
    marks =  (100,90,80,70,60,50)#creating set

print(marks)
    Output
     (100,90,80,70,60,50)
    Set Data

    Set is an unordered collection of elements in which no element is repeated. The elements of sets are separated by a comma and contained within curly braces.

    Example : 
    name_set =  {"subham","shayan","biltu","rocky","ronni"}#creating set

print(name_set)
    Output
    {"subham","shayan","biltu","rocky","ronni"}
