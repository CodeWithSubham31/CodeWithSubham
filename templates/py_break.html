Break Keyword

What is break keyword?

The break keyword is a loop control statement in Python. It is used to stop a loop immediately, even if the loop condition is still true.

Example:
for i in range(10):
    	if i == 7:
    		print("7 founded")
    		break
Output : 
0
1
2
3
4
5
6
7
7 founded

In this code when the for loop value reached to 7 the loop will stop and return 7 found
